📊 Student Grade to Marks Converter

📌 Project Description

This project is a simple Student Grade to Marks Converter developed using Python in a Jupyter Notebook (.ipynb).

It converts student grades (like A, B, C) into corresponding marks and stores the data in an Excel file (.xlsx).

---

🚀 Features

- Input student name and grade
- Convert grades into marks
- Store and read data using Excel file (.xlsx)
- Display student results clearly
- Beginner-friendly project

---

🛠️ Technologies Used

- Python 🐍
- Jupyter Notebook (.ipynb)
- Pandas (for handling Excel data)
- OpenPyXL (for Excel file support)

▶️ How to Run the Project

1. Install required libraries:

pip install pandas openpyxl

2. Open the project in Jupyter Notebook or VS Code

3. Open the notebook file:

main.ipynb

4. Run all cells step by step
